Resources

Linguistics

  • Despite its famously unreliable servers, LingBuzz is still the one-stop shop for open access papers in generative linguistics.
  • Semantics Archive is an open access archive for semantics papers.
  • UCL linguistics publish an annual volume of working papers, which can be found here.
  • I've just started using pst-jtree for drawing syntax trees in LaTeX. The syntax is a little opaque, but it's incredibly flexible.
  • Norbert Hornstein's blog Faculty of Language is essential reading for generative linguists.
  • Gillian Ramchand's blog.

Other

  • Try Haskell in your browser; If you're a semanticist with some interest in learning to code (like me), then this is worth checking out.
  • Along similar lines, Natural Language Processing with Python is a free textbook which provides an excellent way for linguists to get into coding with python.
