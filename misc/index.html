Resources

Linguistics

  • LingBuzz is the one-stop shop for open access papers in generative linguistics. Mitcho Erlewine maintains a public backup as a github repo for those times when the lingbuzz servers are down (which are frequent).
  • Semantics Archive is an open access archive for semantics papers.
  • UCL linguistics publish an annual volume of working papers, which can be found here.
  • Norbert Hornstein's blog Faculty of Language is essential reading for generative linguists.
  • Gillian Ramchand's blog.

LaTeX

  • Forest is currently the most full-featured package for drawing syntax trees, and is under active development.
  • Semantics and Pragmatics provide an excellent BibLaTeX implementation of the Unified Stylesheet for Linguistics Journals. This is what I use for bibliographies.
  • The built-in beamer themes are (to put it mildly) hideous. Metropolis makes beamer presentations look nice.
  • Emacs with AUCTeX is the best LaTeX editor there is. Spacemacs with the LaTeX layer installed provides the same functionality without the hassle.

Software I like

  • Spacemacs, a beautifully designed modular emacs distribution. Highly recommended for as a general purpose text editor.
  • The unofficial gaps fork of i3wm.
  • The ranger file manager.
  • Neovim.
  • Yabar.
  • Mopidy.

My setup

I'm running Fedora 24, and I spend the vast majority of my time in emacs. Here's a nice quote from Neal Stephenson (1998; In the Beginning was the Command Line):

...the engineer-hours that, in the case of Microsoft Word, were devoted to features like mail merge, and the ability to embed feature-length motion pictures in corporate memoranda, were, in the case of emacs, focused with maniacal intensity on the deceptively simple-seeming problem of editing text. If you are a professional writer – i.e., if someone else is getting paid to worry about how your words are formatted and printed – emacs outshines all other editing software in approximately the same way that the noonday sun does the stars. It is not just bigger and brighter; it simply makes everything else vanish.
